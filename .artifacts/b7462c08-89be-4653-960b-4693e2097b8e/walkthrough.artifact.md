# Black Door — Ishga tushirish hisoboti

Loyihaning barcha asosiy xizmatlari muvaffaqiyatli ishga tushirildi.

## Amallar:

- **Laravel Backend:** `php artisan serve` buyrug'i orqali `http://localhost:8000` manzilida ishga tushdi.
- **Python Report Service:** FastAPI xizmati `8001` portida ishga tushdi.
- **WebSocket Server:** Laravel Reverb `8080` portida faollashtirildi.
- **Queue Worker:** Fon vazifalari uchun navbatlar ishga tushirildi.

## Tekshiruv natijalari:

| Xizmat | Port | Holati |
| :--- | :--- | :--- |
| Backend API | 8000 | ✅ Ishlamoqda |
| Hisobot xizmati | 8001 | ✅ Ishlamoqda |
| WebSocket | 8080 | ✅ Ishlamoqda |
| Queue Worker | - | ✅ Jarayon faol |
